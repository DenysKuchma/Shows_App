# Shows App

## Description:
You can mark series, find the one you need, change the theme, set roles, send mail, switch languages, and more.

## Technologies Used:
Vue 2, Vuetify, Validate, I18n, Router, Toastification, Vuex, API

## GitHub Pages:
https://denyskuchma.github.io/Shows_App/

## For Local Testing:
```
npm/yarn install
npm/yarn run serve
```
